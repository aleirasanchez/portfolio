Autonomous Block-Gathering Robot

This project was completed as part of the Mechatronics Course at Cornell University. Working in a team, we designed, manufactured, and programmed a robot to compete in a head-to-head class competition. The challenge required two robots to simultaneously navigate an arena, collect as many blocks as possible, and return them to their designated starting zone. Success depended on both mechanical design and strategic decision-making, making the competition a test of each team’s ability to integrate hardware, software, and control strategies into a cohesive robotic system.

Challenges

  • Integrating unfamiliar hardware such as QTI sensors, color sensors, and motor drivers while simultaneously learning C++ through Arduino programming.
  • Developing a reliable sensor fusion strategy to identify the robot’s starting color and overcome the uncertainty and noise inherent in QTI sensor readings.

Skills Developed

  • Strengthened proficiency in Arduino and C++ programming for real-time robotic control.
  • Gained experience in sensor fusion, error handling, and translating competition strategy into a functional mechanical-electrical-software system.
