ROS-Based Control Architecture for an Underwater Vehicle

Migrated a hybrid MATLAB-ROS control framework for a 6-DOF underwater vehicle into a fully ROS-based system. Developed a custom ROS node for trajectory tracking under the supervision of Professor Louis Whitcomb at Johns Hopkins University.

Challenges

  • Adapting MATLAB algorithms into ROS while learning the ROS ecosystem for the first time.
  • Managing real-time data publishing/subscribing for six thrusters in limited time.

Skills Developed

  • ROS architecture, node creation, and multi-thruster control strategies.
  • System debugging, collaborative research coding practices, and technical documentation.
