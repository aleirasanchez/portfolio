About Me

ROS-Based Control Architecture for an Underwater Vehicle

Migrated a hybrid MATLAB-ROS control framework for a 6-DOF underwater vehicle into a fully ROS-based system. Designed a custom ROS node to handle state derivative calculations, subscribe to thruster data, and publish updated state matrices. Worked under Professor Louis Whitcomb at Johns Hopkins University alongside his PhD research team.

Challenges:

  • Adapting MATLAB algorithms into ROS for the first time required not only translating syntax but also understanding ROS conventions for nodes, topics, and message structures.
  • Balancing limited project time with the complexity of real-time system integration added to the difficulty.

Skills Developed:

  • Strengthened knowledge of ROS architecture, real-time control algorithms, and multi-thruster coordination.
  • Improved debugging strategies, system-level thinking, and collaboration in a high-level research environment.

Autonomous Vehicle Design and Control

Built and programmed autonomous racing robots using Python and C++ for the Fast Robots Course at Cornell Universtiy. Focused on localization with IMUs and time-of-flight sensors, real-time motor control, and trajectory optimization.
Project Portfolio: https://aleirasanchez.github.io/aleira.sanchez/

Challenges:

  • Integrating multiple sensor types (IMUs, time-of-flight sensors) for accurate localization in real-time.
  • Developing efficient control algorithms to manage high-speed trajectory optimization while preventing instability or collisions.

Skills Developed:

  • Strengthened expertise in Python and C++ for embedded control and robotics applications.
  • Gained experience in sensor fusion, real-time motor control, and trajectory planning for autonomous systems.

Haptic Feedback Device Research

Researched and characterized eccentric rotating mass and voice coil motors for movement compensation devices. Designed MATLAB applications for automated data collection with DAQ systems and improved device circuitry for human-subject testing.

Challenges:

  • Characterizing the performance of different motors (eccentric rotating mass vs. voice coil) for accurate movement compensation in human-subject experiments.
  • Designing MATLAB applications for automated, reliable data collection while ensuring synchronization with DAQ systems.

Skills Developed:

  • Enhanced proficiency in MATLAB programming, data acquisition, and motor control for haptic systems.
  • Developed skills in experimental design, circuit improvement, and human-subject testing for research applications.
